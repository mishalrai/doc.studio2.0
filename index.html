LOGO

To update the site logo, we have two options: update the image or the SVG. If both are uploaded, the system will prioritize the SVG logo. If the SVG logo is not found, it will then search for the image logo.

Here are the steps to update the logo:

  1. Go to Appearance in side menu > Customize

    Cutomize

  2. Click on Site Identity option from the availabel option

  3. In this step, you will find the option to upload images and SVG files.

    Click on select logo to upload imge

    Please check the link for more details about the steps to upload an image How to upload Image

    Paste the SVG logo code here
  4. After updating one of them field click on publish button to update the logo
Recommended width of logo is 80px. and don't forget to add width attribute in the SVG

Menu

We are using the same menu for both desktop and mobile, so any updates made from the admin panel will affect both platforms.

Here are the steps to update the Menu:

  1. Go to Appearance in side menu > Menu

    Main menu

  2. Expand the Custom links and add the URL and link text and click Add to menu

  3. Recently added menu items will appear at the bottom of the menu list, similar to contacts in the image below. We can also arrange the menu order.

    Main menu

  4. As our site has a single-page design, the menu links only scroll to particular sections instead of redirecting to another page. Therefore, we need to pass the section ID in the URL as a hash value, as shown in the picture below.

    Main menu

Available section hash values

#about, #team, #projects, #latest-blogs, and #contacts
